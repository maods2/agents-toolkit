# Playbook: Research Feature

## Use when

A feature combines software implementation with research, experiments, evaluation, or paper updates.

## References

- Principles: `principles/software-engineering.md`, `principles/testing.md`
- Skills: `skills/workflow-discovery/`, `skills/python/`, `skills/mlops/`, `skills/pytorch/`, `skills/experiment-design/`, `skills/scientific-writing/`

## Workflow

1. Discovery: understand the workflow, constraints, data, and existing implementation.
2. Literature Review: summarize relevant prior work and reusable methods.
3. Experiment Design: define hypothesis, datasets, baselines, metrics, controls, and logging.
4. Implementation: build the smallest reproducible experiment or feature slice.
5. Evaluation: compare results against the protocol and record limitations.
6. Paper Update: update methods, results, discussion, and reproducibility notes.
